User Guide

Just paste once — all selected photos instantly appear.

After clicking “Copy Filenames” in the dashboard, use this quick guide to locate the original files on your computer — via Finder, File Explorer, or directly inside Lightroom.

Why are the filenames extensionless?

These aren't corrupted or cut-off filenames — in fact, this is precisely what makes the process work across all file types simultaneously.

What is copied

Only the filenames, space-separated, without trailing ".jpg" or ".cr2". This format is intentionally built to be pasted directly into search bars on any computer.

Why this helps

A name-based search will locate all files with matching names, regardless of extension. So if clients select from a JPG preview, the original RAW files (.CR2, .NEF, .ARW, etc.) are filtered automatically — without you searching one by one.

Choose the method for your device

All four methods below use the exact same filename list — just choose the tab matching the software or OS you normally use.

📁 Perfect for gathering selected RAW + JPG files into a new folder before importing to any editing software.

Open original photo folder

Open the folder storing all the shots (RAW and JPG) from that photo session.

Click the search bar

Click the magnifying glass icon in the top-right corner of the Finder window.

Paste the filename list

Paste using Cmd + V. Finder will instantly display every file whose name matches the list — both RAW and JPG of the same name will appear.

Select all results

Press Cmd + A to select all displayed files, then Cmd + C to copy.

Collect in a new folder

Create a new empty folder, open it, then paste using Cmd + V. All selected files are now gathered in one place.

Note Note: The Finder search bar has a limit of about 2,048 characters or ±99 filenames in a single paste. If the client list is longer, split and paste in batches.
📁 Just like Finder, the end result is a new folder containing the files chosen by the client, ready to edit or import.

Open original photo folder

Open the folder storing all the shots from that photo session.

Click the search box

Click the "Search" box in the top-right corner of the File Explorer window.

Paste the filename list

Paste using Ctrl + V. Explorer automatically filters and displays matching files, including RAW and JPG of the same name.

Select all results

Press Ctrl + A, then Ctrl + C to copy all displayed files.

Collect in a new folder

Create a new empty folder, open it, then paste using Ctrl + V.

Note Note: The Explorer search bar has a tighter limit of about 259 characters per paste. For long lists, paste and process in batches (e.g. 15–20 names per batch).
⚡ The fastest way — no need to duplicate physical files; filter directly within your Lightroom Catalog.

Ensure photos are imported

Open Lightroom Classic and make sure the folder or catalog containing this session's photos is imported.

Select source photos

Click All Photographs in the Catalog panel on the left, or select the specific project folder, so the search queries all possible session photos.

Open Library Filter Bar

Press the \ key or click the filter area above the grid view, then select the Text category.

Set to Filename / Contains

Change the first dropdown to Filename and the second to Contains — do NOT choose "Contains All", as that searches for files containing all words combined.

Paste the filename list

Click the search input box, then paste using Cmd/Ctrl + V. All matching photos will instantly appear in the grid, both RAW and JPG.

Save as a Collection

Press Cmd/Ctrl + A to select all results, then create a new Collection to easily access them later during editing.

Penting Important: Filenames in the list must be separated by spaces or commas — not one per line, as the filter won't work with newlines. The dashboard copy button formats this automatically.
Info Please Note: This feature is only available in Lightroom Classic. In the cloud-based Lightroom (non-Classic), filenames must be searched one by one.
🎯 For editing in Capture One instead of Lightroom — this is a native Capture One feature.

Open folder or collection

Open the folder/collection in Capture One containing all photos from this session.

Open Select by File Name List

From the menu bar, choose Select → Select by → File Name List…

Paste the list

Paste the filename list into the text box that appears.

Set separator and extension

Choose Space as the separator, then check Ignore file extension so RAW and JPG of the same name are selected.

Click OK

All matching files are instantly selected in the Capture One browser, ready to organize or edit.

Lightroom direct filter, or collect in a folder first?

Both methods are perfectly valid — choose the one that fits your workflow.

Filter directly in Lightroom

Fastest

  • Create a Collection and start editing immediately
  • Requires photos to be already imported in Lightroom Classic
  • Does not apply to cloud-based Lightroom

Most Flexible

  • Pasting list has character limits per paste

Frequently Asked Questions

Why is the search result empty after pasting?

For Lightroom, make sure names are separated by spaces or commas, not one per line. For Finder/Explorer, ensure the folder you opened actually contains the original files, and try pasting a smaller batch if the list is long.

Why do RAW files show up when I copied filenames from a JPG gallery?

Because the search operates based on filenames regardless of extension. All files with the same name — both .jpg and .CR2/.NEF/.ARW — will be found. This is intentional so that your original RAW files are automatically gathered.

My client's filename list is extremely long, what should I do?

Split the list into several parts and paste in batches — especially in Windows Explorer which has a tighter limit (±259 characters). Finder holds more (±99 files), and Lightroom Classic can handle much longer lists in a single paste.

I don't use Lightroom or Capture One, can I still use this?

Yes. Simply use the Finder (Mac) or Explorer (Windows) method to gather files into a new folder, then open that folder in whatever editing software you use.
